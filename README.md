# 👗 Bharat Fashion AI

An AI-powered Fashion Recommendation System built using React, FastAPI, and Python.

## Overview

Bharat Fashion AI provides personalized fashion recommendations based on:

* Gender
* Occasion
* City
* Budget
* Product Ranking Score

The system helps users discover fashion products tailored to their preferences and requirements.

## Features

* Personalized fashion recommendations
* City-based filtering
* Occasion-based outfit suggestions
* Budget-aware recommendations
* Rating-based ranking system
* Responsive modern UI
* REST API architecture

## How to Run

### Backend

```bash
cd backend
venv\Scripts\activate
python -m uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
